KAN vs. ANN: A Comparative Study for Protein and Succinylation Prediction
Overview
This project explores the Kolmogorov-Arnold Network (KAN) and compares it to a traditional Artificial Neural Network (ANN) for protein structure prediction and succinylation site prediction. The research focuses on understanding the differences in model performance, interpretability, and computational efficiency.
